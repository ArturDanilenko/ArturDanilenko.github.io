Projects

Coin-picking Robot

Objective: Given an enclosed area, robot is supposed to autonomously search for the coins and collect them once encountered.

Implementation: I was responsible for the robot's software. Since we were working with a new microcontroller, one of the most challenging parts involved finding appropriate registers in order to set up IO pins with a desired functionality which involved going through extremely large datasheets. Secondly, interupts frequency was very sensitive to the amount of instructions processed upon being triggered, hence required very detailed and thorough optimization in order to function accurately.

Reflow Oven Controller

Objective: Program and assemle a controller which runs an oven through reflow soldering process with an ability to alter the temperature and time setting for each state.

Implementation: Being assigned to software team, my task was to implement a specific state machine which controls oven using PWM module. The challenging aspect happened to be scheduling of the states and synchronization of multiple timers. The problem was solved by using a singular timer and making multiple sub-timers based on the main one. Once the project guidelines were met, code had to be heavily optimized as assembly structure didn't allow to be easily managed while the main goal isn't met.

Line-following Robot

Objective: Provided with a track, robot had to quickly follow a black line while appropriately reacting to obstacles.

Implementation: Robot had five grayscale sensors that could accurately identify shade of gray that robot is passing if it's adequetly callibrated. This happened to be an issue as the sensitivity would fluctuate depending on the dirt level of the surface and battery life. To overcome this, we designed an algorithm which allowed robot to autocallibrate itself if placed on a trial track. This was accomplished by taking measurements of his actual readings and finding a mid-point between where he sensed a line and the rest of the environment.
